Формулировка задания: Разработать стандартизированное (шаблонное) решение, для оповещения пользователя по совершаемым операциям по его счету. 
Для оповещения используется объект, описывающий взаимодействие пользователя и банка в платформе TranzAxis - контракт. 
Контракт нотификации содержит несколько типов подписок (тип подписки - событие, источник для нотификации)
Подписка содержит ряд шаблонных сообщений, с параметрами для их применения (регулирование применения)
На основе транзакции выбирается одно шаблонное сообщение, теги в нем заменяются на информацию из транзакции.
